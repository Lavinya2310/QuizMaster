About Quiz Master

Quiz Master is a simple yet powerful web-based quiz platform built for students and educators.

Key Features

  • Admin panel to create subjects, chapters, quizzes & questions
  • User registration & secure login
  • Timed MCQ quizzes with instant scoring
  • Performance dashboards & charts
  • Built with Flask, SQLite, Bootstrap & Chart.js
